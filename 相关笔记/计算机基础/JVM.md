# 概述

JVM 包含两大部分内容 ： 内存管理和类加载机制

- 因为Java自动管理内存，所以我们要弄明白JVM是怎么管理内存的
- Java是纯面向对象语言，万物皆类，我们想要弄明白，类（也就是我们写的代码）是如何从代码，到加载到JVM，再到运行的
