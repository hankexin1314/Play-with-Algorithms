- 位操作优先级很低，需要加括号

#### [剑指 Offer 15. 二进制中1的个数](https://leetcode-cn.com/problems/er-jin-zhi-zhong-1de-ge-shu-lcof/)

```java
public int NumberOf1(int n) {

    int count = 0;
    while(n != 0) {
        if((n & 1) == 1) // 这里必须加括号
            count ++;
        n = n >>> 1;
    }

    return count;
}
```
